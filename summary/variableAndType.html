변수
변수란, 하나의 값을 저장할 수 있는 메모리 공간이다.
하나의 변수에는 하나의 타입의 값만 저장할 수 있다.
변수는 초기화 되어야 읽을 수가 있고, 초기화 되지 않은 변수는 읽을 수가 없다.
변수는 선언된 블록 내에서만 사용이 가능하다.
자바 변수 명명
관례적으로 첫문자는 영어 소문자로 시작, 다른 단어가 붙을 경우 첫 문자를 대문자로 한다.
어떤 값을 저장하고 있는지 쉽게 알 수 있도록 변수 이름을 지어주는 것이 좋다. 충분히 길어도 상관 없다.

소스 코드 내에서 직접 입력된 값

이스케이프(escape) 문자
역슬래쉬(\)가 붙은 문자 리러털

데이터 타입

데이터 타입
타입에 따라 저장할 수 있는 값의 종류와 범위가 달라진다.
선언시 주어진 타입은 변수를 사용하 도중에 변경할 수 없다.
타입 변환
데이터 타입을 다른 데이터 타입으로 변환하는 것
자동(묵시적)타입 변환, 강제(명시적) 타입 변환
자동(묵시적)타입 변환(Promotion)
프로그램 실행 도중에 자동적으로 타입 변환이 일어나는 것을 말한다.
서로 다른 타입의 피연산자(operand)들이 연산될때 작은 타입의 피연산자가 큰 타입으로 자동타입 변환된 후 연산된다.
작은 크기를 가지는 타입이 큰 크기를 가지는 타입에 저장될 때 발생한다.
int->double, byte->int
강제 타입 변환
큰 크기의 타입의 데이터의 일부분을 작은 크기의 타입의 공간에 넣는다.
데이터의 손실이 일어난다.
작은크기 타입 = (작은 크기 타입)큰 크기타입
